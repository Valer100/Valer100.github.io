Volume Labeler
A simple tool for changing the label and the icon of a volume in Windows. It makes these changes by creating an autorun.inf file (or edits the existing one) on the volume you want to change its label and icon. O unealtă simplă pentru schimbarea etichetei și pictogramei unui volum pe Windows. Aceasta realizează aceste modificări prin crearea unui fișier autorun.inf (sau îl editează pe cel existent) pe volumul pe care vrei să-i schimbi eticheta și pictograma.

 Features

 Caracteristici
  • When selecting a volume, it checks if autorun.inf is present on it. It then retrieves its actual label and icon, so you don't have to type the same label if you only want to change the volume's icon or select the same icon if you only want to change the volume's label.
  • Atunci când selectezi un volum, aplicația verifică dacă autorun.inf este prezent pe el. Apoi obține eticheta și pictograma sa actuală ca să nu mai trebuiască să scrii aceeași etichetă dacă vrei doar să schimbi pictograma volumului sau să selectezi aceeași pictogramă dacă doar vrei să schimbi eticheta volumului.

  • Multiple icon options: default icon, custom icon (.ico or from an .exe, .dll or .icl file) or icon from image (converts the selected image to an .ico file).
  • Opțiuni multiple pentru pictogramă: pictograma prestabilită, pictogramă personalizată (.ico sau dintr-un fișier .exe, .dll sau .icl) sau pictogramă din imagine (convertește imaginea selectată într-un fișier .ico).

  • Refresh volume information in the File Explorer after applying the changes (requires running the app with administrator rights and not using the volume; doesn't work with the system volume)
  • Reîmprospătează informațiile volumului în File Explorer după aplicarea modificărilor (necesită rularea aplicației cu drepturi de administrator și neutilizarea volumului; nu funcționează cu volumul de sistem)

  • Option to hide the autorun.inf file and the vl_icon folder (the icon is stored in that folder).
  • Opțiune pentru ascunderea fișierului autorun.inf file și folderului vl_icon (pictograma este stocată în acel folder).

  • Option to backup the autorun.inf file for easily restoring it later if something wents wrong or you want to revert your changes to a previous point.
  • Opțiune pentru a face backup fișierului autorun.inf pentru a-l putea restaura cu ușurință mai târziu dacă ceva nu merge cum trebuie sau vrei să-ți anulezi modificările până la un punct anterior.

  • Option to get rid of all customizations (including the ones not made by Volume Labeler).
  • Opțiune pentru eliminarea tuturor personalizărilor (inclusiv cele care nu au fost făcute de Volume Labeler).

  • Entry in the volumes' right click context menu for easily customizing volumes (available only in the classic context menu at the moment).
  • Intrare în meniul contextual al volumelor pentru personalizarea volumelor cu ușurință (momentan disponibilă doar în meniul contextual clasic).

  • Light and dark themes and localization support.
  • Temă luminoasă și întunecată și suport pentru localizare.
